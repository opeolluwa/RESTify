## Installation
Clone from GitHub

```bash
  git clone https://github.com/restify
  cd  restify 
  npm.i
```
    
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`JWT_KEY`
`DB_ACCESS_KEY`
`DB`SCHEMA`
`DB_USER`
`DB_HOST`


## Features

- UserAuthentication
- News letter
- Contact mails
